# CodeLabAPIPessoa

API de Pessoa do CodeLab.

## Development

```bash
docker-compose up --build
```

## Testing

Attach to the running container and run the tests.

### Unit

```bash
    npm run test:cov
```

### E2E

Attach to the running container and run the tests.

```bash
    npm run test:e2e:cov
```

For not generating coverage reports, run without the `:cov` suffix.
